# 🛒 POS-System

ئەمە سیستەمێکی خاڵی فرۆشتن (Point of Sale)ـە کە بە تەکنەلۆژیای مۆدێرن دروستکراوە بۆ ئاسانکاری و خێراکردنی کارەکانی فرۆشتن، بەڕێوەبردنی کۆگا و شیکاری فرۆشتنەکان.

## 🛠 تەکنەلۆژیاکان (Tech Stack)
پڕۆژەکە لەم تەکنەلۆژیایانە پێکهاتووە :

- **Electron.js**: بۆ دروستکردنی ئەپلیکەیشنی دیسکتۆپ.
- **React.js**: بۆ ڕووکاری بەکارهێنەر (UI).
- **SQLite3**: بۆ پاشەکەوتکردنی داتاکان (داتابەیسی `POS_shop.db`).
- **Tailwind CSS**: بۆ دیزاین و ستایل.
- **Vite**: بۆ خێرایی و کارپێکردنی پڕۆژەکە.

## 🚀 تایبەتمەندییەکان
- **بەڕێوەبردنی فرۆشتنەکان**: تۆمارکردنی داهاتی گشتی و ژمارەی فەرمانەکان (Orders).
- **دابەشکراوی بەشی چێشتخانە**: پشتگیری بۆ خشتەی دانیشتن و پلانی نهۆم (Floor Plan).
- **شیکاری (Analytics)**: بوونی بەشێک بۆ ڕاپۆرت و شیکاری فرۆشتنەکان.
- **بەڕێوەبردنی کۆگا**: چاودێریکردنی کاڵاکان و بەرهەمە بەسەرچووەکان.

## 📂 پێکهاتەی پڕۆژەکە
- `src/`: کۆدی سەرەکی ڕیاکت (React Components, Hooks, Stores).
- `main.cjs`: فایلی سەرەکی ئەلیکترۆن بۆ بەڕێوەبردنی پەنجەرە و پەیوەندی لەگەڵ داتابەیس.
- `aram_shop.db`: داتابەیسی سەرەکی پڕۆژەکە.

## 💻 چۆنیەتی کارپێکردن
بۆ کارپێکردنی پڕۆژەکە، دڵنیا ببەرەوە کە Node.jsـت هەبێت، پاشان لە تێرمیناڵەکەتدا ئەم فەرمانانە بنووسە:

```bash
# دابەزاندنی پێداویستییەکان
npm install

# کارپێکردنی پڕۆژەکە
npm run dev
